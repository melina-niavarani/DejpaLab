# DejpaLab - آزمایشگاه دژپا

وبسایت دو زبانه (فارسی/انگلیسی) برای آزمایشگاه دژپا با استفاده از Next.js

## ویژگی‌ها

- ✅ دو زبانه (فارسی و انگلیسی)
- ✅ طراحی واکنش‌گرا (Responsive)
- ✅ استفاده از Next.js 14 با App Router
- ✅ پشتیبانی از RTL برای زبان فارسی
- ✅ رابط کاربری مدرن و حرفه‌ای
- ✅ بهینه‌سازی برای SEO

## نصب و راه‌اندازی

### پیش‌نیازها

- Node.js 18+ 
- npm یا yarn

### نصب

```bash
npm install
```

### اجرای پروژه در حالت توسعه

```bash
npm run dev
```

سپس مرورگر را در آدرس [http://localhost:3000](http://localhost:3000) باز کنید.

### ساخت نسخه Production

```bash
npm run build
npm start
```

## ساختار پروژه

```
├── app/
│   ├── [locale]/          # صفحات با پشتیبانی از زبان
│   │   ├── page.tsx       # صفحه اصلی
│   │   ├── services/      # صفحه خدمات
│   │   ├── about/         # صفحه درباره ما
│   │   └── contact/       # صفحه تماس
│   ├── layout.tsx         # Layout اصلی
│   └── globals.css        # استایل‌های全局
├── components/            # کامپوننت‌های قابل استفاده مجدد
│   ├── Navigation.tsx     # منوی ناوبری
│   └── Footer.tsx         # فوتر
├── messages/              # فایل‌های ترجمه
│   ├── fa.json           # ترجمه فارسی
│   └── en.json           # ترجمه انگلیسی
├── i18n/                  # تنظیمات بین‌المللی‌سازی
└── public/                # فایل‌های استاتیک
```

## زبان‌ها

- فارسی (fa) - زبان پیش‌فرض
- انگلیسی (en)

برای تغییر زبان، از دکمه تغییر زبان در منوی ناوبری استفاده کنید.

## صفحات

- **صفحه اصلی**: معرفی آزمایشگاه و خدمات اصلی
- **خدمات**: لیست کامل خدمات آزمایشگاهی
- **درباره ما**: اطلاعات درباره آزمایشگاه، ماموریت و چشم‌انداز
- **تماس با ما**: فرم تماس و اطلاعات ارتباطی

## تکنولوژی‌های استفاده شده

- **Next.js 14**: فریمورک React
- **TypeScript**: برای type safety
- **Tailwind CSS**: برای استایل‌دهی
- **next-intl**: برای بین‌المللی‌سازی
- **Lucide React**: برای آیکون‌ها

## سفارشی‌سازی

### تغییر رنگ‌ها

رنگ‌های تم را می‌توانید در فایل `tailwind.config.ts` تغییر دهید.

### افزودن محتوا

محتواهای متنی را می‌توانید در فایل‌های `messages/fa.json` و `messages/en.json` ویرایش کنید.

برای سوالات و پشتیبانی، با تیم توسعه تماس بگیرید.
